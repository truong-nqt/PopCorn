# API Keys
Để chạy ứng dụng này trên máy của bạn, bạn phải cấp API KEY từ [The Movie DB](https://www.themoviedb.org/) 
và đặt nó vào thư mục strings.xml của bạn.
```
<string name="MOVIE_DB_API_KEY">YOUR API KEY HERE</string>
```
